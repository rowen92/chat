# Chat
###### Simple CLI chat server. Clients can connect and send messages.

### Requirenments
* NodeJS >= 4.x

### Install
```sh
$ git clone git@github.com:rowen92/chat.git
$ npm install
```

### Start server
```sh
$ node app/server
```

### Start client
```sh
$ node app/client
```

### Testing
```sh
$ npm test
```
